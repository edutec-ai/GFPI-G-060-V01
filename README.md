## 📋 Descripción

Este repositorio contiene recursos para el **Desarrollo Curricular** del SENA, basados en la nueva **Guía de Desarrollo Curricular (Junio 2026)**.

El recurso principal es una **landing page interactiva** que permite:

- 📄 Descargar la **guía anterior** (GFPI-G-012-V1)
- 📄 Descargar la **nueva guía** (GFPI-G-060 V01)
- 📊 Descargar la **presentación en Power Point** sobre la cascada
- 🎙️ Descargar el **podcast** sobre desarrollo curricular
- 🌊 Acceder a la **visualización interactiva "Cascada"** con pestañas y dashboards
